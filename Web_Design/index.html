Lesson 2: Div, Span, Id's

You can use a div to contain a section of information to structure out your website to make it easier to style later.

A div will move the contents down one line of code.

You can use a span to contain a section of information to structure out your website to make it easier to style later.

A Span will keep the contents on the same line of code.

Ex. If you insert a div between Lesson 2: and Div, Span, Id's then the Div, Span, Id's would appear on the line below Lesson 2.

If you used span instead it would be as it is in bold at the top of the page.

You can add an ID to your div like div id=course info to make the code easier to read and sytle later on.
